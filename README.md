# Student Result System (Python)

## About

This program calculates a student's total marks, average, grade, and pass/fail result.

## Features

* Calculates total and average
* Assigns grade
* Determines pass/fail
* Handles invalid input

## Tech Used

* Python

## How to Run

Run the Python file and enter student details when prompted.

## Future Improvements

* Add multiple students support
* Save results to file
